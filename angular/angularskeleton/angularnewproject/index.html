Angular new project

Create and install packages

npx ng new <projectname>

-> Routing yes -> CSS

Install material

cd <projectname>
ng add @angular/material

Homescreen

Create the homescree component

ng g c screens/homeScreen

Import that component from app-routing.module.ts and check the the Routes , RouterModule are imported and array created.

import { HomeScreenComponent } from './screens/home-screen/home-screen.component';
import { RouterModule, Routes } from '@angular/router';

const routes: Routes = [];

Check that the app.module imports the app-routing-moudule Create a rout in app-routing.module

const routes: Routes = [
  { path: 'home-screen', component: HomeScreenComponent },
];

The order of routes is important because the Router uses a first-match wins strategy when matching routes, so more specific routes should be placed above less specific routes. List routes with a static path first, followed by an empty path route, which matches the default route. The wildcard route comes last because it matches every URL and the Router selects it only if no other routes match first.

Update the app.component.html

<h1>Angular Router App</h1>
<header>
  <nav>
    <ul>
      <li><a routerLink="/home-screen" routerLinkActive="active" ariaCurrentWhenActive="page">First Component</a></li>
    </ul>
  </nav>
</header>
<main>This is the main</main>
<footer>This is the footer</footer>

<!-- The routed views render in the <router-outlet>-->
<router-outlet></router-outlet>

On the app.module.css setup the high of the main

main {
    min-height: 80vh;
}
