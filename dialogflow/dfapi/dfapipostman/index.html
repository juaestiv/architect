Dialogflow API Postman

  1. Open Postman
  2. Create a Post/get Request
  3. From Authorization folder choose OAUTh 2.0
  4. Autenticate with user owner and password
  5. Click on use token

Example of GET requests List Agents This is an example of list agentes GET https://global-dialogflow.googleapis.com/v3/projects/hospitality-demo-361210/locations/global/agents/

Example of POST Request

Agent Intents belongs to sessions POSR https://cloud.google.com/dialogflow/cx/docs/reference/rest/v3/projects.locations.agents.environments.sessions
