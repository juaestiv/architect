Terraform Staging

Intro

I am going to use a terraform project to deploy resources in three environments, dev, staging and production. There are several alternatives. First one will be using several directories with variables

Option 1.

Step 1. Create a folder tree

Create a folder tree as follows

yourproject
│   ├── modules
│   ├── prod
│   └── staging

We can create three VPC one per environment. To do dat first we are going to create a moudule for vpc under modules folder. We can create a subfolder called vpc_module and create main and variable tf files

% tree
.
├── modules
│   └── vpc_module
│       ├── main.tf
│       └── variables.tf
├── prod
└── staging
