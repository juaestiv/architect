Hugo Layouts

Layout

Index.html

First layout to pay attention is the one under the folder layouts/index.html If you need to edit that layout yu can make a copy from themes/whatevertheme/layouts/index.html folder in layouts/index.html Let’s say that your content is shown in 3 columns (size of 4 each one) to 4 Columns (size of 3 each) you can chang as follows

 <!-- topic-item -->
        {{ range (where .Site.Pages "Type" "docs") }}
          <div class="col-lg-3 col-sm-6 mb-4">

Custom CSS

You can use your own css files to override de css in use.

  1. edit config.toml
custom_css = ["css/foo.css", "css/bar.css"]

In static folder create the css directory and the files

Then on the partial that have the head tab

  <!-- ** CSS Plugins Needed for the Project ** -->
  {{ range .Site.Params.custom_css -}}
  <link rel="stylesheet" href="{{ . | absURL }}">
  {{- end }}

for example in layout/partials/head.html

It is important where to put the code. As there are a lot of css I put it at the end of the head

Custom fonts

  1. config.toml
custom_font = ["http://fonts.cdnfonts.com/css/vielkalahizo"]

  {{ range .Site.Params.custom_font -}}
  <link rel="stylesheet" href="{{ . | absURL }}">
  {{- end }}
  1. edit your custom css
h3 {
    font-family: Vielkalahizo;
    color: rgb(150, 93, 150)
  }
