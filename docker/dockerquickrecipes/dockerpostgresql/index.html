PostgreSQL

Recall

//list all images
docker image ls 
docker ps

//shows runing containers
docker container ls 

//shows all containers
docker container ls  -a

Install psql client

brew install postgresql

Create a docker container with a yaml file

touch docker-compose.yml

Edit

version: "3"
services:
  db:
    image: postgres
    restart: unless-stopped
    container_name: "my_postgres"
    ports:
      - "5432:5432"
    environment:
      POSTGRES_USER: postgres
      POSTGRES_PASSWORD: SuperSecret!23
    volumes:
      - .docker-data/postgres:/var/lib/postgresql/data
volumes:
  my_dbdata:

To run the container

docker compose up -d

List runing containes

docker container ls

To stop the postgresl server

docker composer down

This delete the containers but not the image

To access the docker

docker exec -it my_postgres psql -U postgres

Then connect to postgres

psql -h 127.0.0.1 -U postgres

list tables

\l

List the local volume

docker volume ls
